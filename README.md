# American Express Card Finder

A web application to search and filter American Express credit cards by rewards, fees, and card type.

## Live Site

The application is hosted on GitHub Pages and can be accessed at:

**https://waaron5.github.io/amex-card-finder/**

## Features

- Search cards by name
- Filter by card type (Personal/Business)
- Filter by rewards category (Travel, Dining, Groceries, etc.)
- Filter by annual fee range

## How to Use

Simply visit the [live site](https://waaron5.github.io/amex-card-finder/) and use the filters to find the right American Express card for your needs.

## Local Development

To run locally, simply open `index.html` in your web browser.
